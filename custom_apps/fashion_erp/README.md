# Fashion ERP

`fashion_erp` is a custom ERPNext app for women's apparel ecommerce and garment manufacturing.

Current scope in this repository:

- app packaging skeleton
- phase-1 master-data dictionaries
- idempotent seed patch for color and size dictionaries

Repository path:

- source in this repo: `custom_apps/fashion_erp`
- runtime path inside a bench: `apps/fashion_erp`
